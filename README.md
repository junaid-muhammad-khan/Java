## Overview

This repository contains my Java coursework, practice exercises, labs, projects, and examples completed during my university Java courses.

The content includes a combination of:

- Lecture practice code
- Laboratory assignments

- Exception handling
- User input and output
- Problem-solving and algorithm implementation
